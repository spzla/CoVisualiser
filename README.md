# CoVisualiser
A client-side administration tool designed to ease the workflow of [CoreProtect](https://github.com/PlayPro/CoreProtect).

## Downloads
You can currently download it on the [GitHub Releases](https://github.com/spzla/covisualiser/releases) page.

## How does it work?
It scrapes the chat for lookup results after using a command.<br>
You can use it by adding one of the following to your command: `#cv`, `#covis`, `#covisualiser`, `#covisualizer`.

## Examples
- `/co l a:-block i:diamond_ore t:1d u:spzla #cv`
- `/co l a:+block i:netherite_block t:20w u:spzla #covisualiser`

## Features
- Reduced chat spam due to moving CoreProtect lookups into a neat interface

## Planned features
- In-world block rendering
- Collapse results into veins

## Dependencies
- [YetAnotherConfigLib](https://github.com/isXander/YetAnotherConfigLib)
- [Fabric API](https://github.com/FabricMC/fabric-api)
